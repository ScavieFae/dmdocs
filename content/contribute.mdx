## Report an Issue

Found an error in the rules text, a missing spell, or a broken link?

Open an issue on GitHub: **[github.com/ScavieFae/dmdocs/issues](https://github.com/ScavieFae/dmdocs/issues)**

When reporting content issues, please include:
- The page URL
- What's wrong (quote the incorrect text if applicable)
- What it should say (with SRD page reference if you have it)

## Contribute Code

The site is built with Next.js and Fumadocs. Content is in MDX files.

1. Fork the repo
2. Make your changes
3. Open a pull request

See the [README](https://github.com/ScavieFae/dmdocs) for setup instructions.

## Content Guidelines

All content should be verifiable against the [official SRD 5.2.1](https://www.dndbeyond.com/srd). We don't include:

- Content from the full Player's Handbook that isn't in the SRD
- Homebrew or house rules
- Third-party content

If you're unsure whether something belongs, open an issue to discuss.
