The SRD includes a wide variety of creatures for encounters, from lowly beasts to legendary dragons.

## Monster Types

| Type | Description |
|------|-------------|
| Aberration | Utterly alien beings |
| Beast | Natural animals |
| Celestial | Creatures of the Upper Planes |
| Construct | Animated objects and golems |
| Dragon | Winged, reptilian creatures |
| Elemental | Creatures of the elemental planes |
| Fey | Creatures of the Feywild |
| Fiend | Creatures of the Lower Planes |
| Giant | Human-like creatures of great size |
| Humanoid | Bipedal peoples |
| Monstrosity | Unnatural creatures |
| Ooze | Amorphous creatures |
| Plant | Vegetable creatures |
| Undead | Once-living creatures animated by magic |

## Challenge Rating

Challenge Rating (CR) indicates relative threat level:

| CR | XP | Typical Threat |
|----|----|----|
| 0 | 0-10 | Harmless |
| 1/8 - 1/4 | 25-50 | Minor threat |
| 1/2 - 1 | 100-200 | Single enemy for low levels |
| 2-4 | 450-1,100 | Challenging for low levels |
| 5-10 | 1,800-5,900 | Mid-tier threats |
| 11-16 | 7,200-15,000 | High-tier threats |
| 17-20 | 18,000-25,000 | Legendary creatures |
| 21+ | 33,000+ | Near-godlike threats |
