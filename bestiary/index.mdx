The Bestiary contains stat blocks and descriptions for creatures found in the D&D 5e System Reference Document.

## Using This Reference

Each monster entry includes:

- **Stat block** with all mechanical information
- **Description** of the creature's nature and behavior
- **Tactics** for how the creature fights

## Monster Categories

Monsters are organized alphabetically. Use the sidebar to navigate, or search for specific creatures.
