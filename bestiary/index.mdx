# Bestiary

Browse monsters organized by creature type. Each type represents a different category of being in the D&D multiverse.

## Creature Types

| Type | Description |
|------|-------------|
| **[Aberration](/bestiary/aberration)** | Utterly alien beings from beyond the planes |
| **[Beast](/bestiary/beast)** | Natural animals and giant versions thereof |
| **[Celestial](/bestiary/celestial)** | Beings of the Upper Planes, often divine servants |
| **[Construct](/bestiary/construct)** | Magically created creatures and automatons |
| **[Dragon](/bestiary/dragon)** | Ancient scaled beings of immense power |
| **[Elemental](/bestiary/elemental)** | Creatures of pure elemental essence |
| **[Fey](/bestiary/fey)** | Magical beings tied to nature and the Feywild |
| **[Fiend](/bestiary/fiend)** | Evil beings from the Lower Planes |
| **[Giant](/bestiary/giant)** | Towering humanoids of tremendous strength |
| **[Humanoid](/bestiary/humanoid)** | People of various species and professions |
| **[Monstrosity](/bestiary/monstrosity)** | Unnatural creatures with strange origins |
| **[Ooze](/bestiary/ooze)** | Mindless, gelatinous predators |
| **[Plant](/bestiary/plant)** | Sentient vegetation and fungal creatures |
| **[Undead](/bestiary/undead)** | Spirits and reanimated corpses |

## Reading a Stat Block

- **AC** - Armor Class, the target number to hit
- **HP** - Hit Points, with average and dice formula
- **Speed** - Movement in feet per round
- **CR** - Challenge Rating, a rough measure of threat level
