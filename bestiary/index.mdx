## Creature Types

| Type | Description |
|------|-------------|
| **Aberration** | Alien beings from beyond the planes |
| **Beast** | Natural animals and giant versions |
| **[Celestial](/bestiary/celestial)** | Divine servants of the Upper Planes |
| **[Construct](/bestiary/construct)** | Magically created automatons |
| **[Dragon](/bestiary/dragon)** | Ancient scaled beings of immense power |
| **[Elemental](/bestiary/elemental)** | Creatures of pure elemental essence |
| **[Fey](/bestiary/fey)** | Magical beings of nature and the Feywild |
| **[Fiend](/bestiary/fiend)** | Devils and demons of the Lower Planes |
| **Giant** | Towering humanoids of great strength |
| **Humanoid** | People of various species |
| **Monstrosity** | Unnatural creatures with strange origins |
| **Ooze** | Mindless, gelatinous predators |
| **Plant** | Sentient vegetation and fungi |
| **Undead** | Spirits and reanimated corpses |

## Stat Block Quick Reference

- **AC** — Armor Class (target to hit)
- **HP** — Hit Points (average and dice formula)
- **Speed** — Movement per round in feet
- **CR** — Challenge Rating (threat level and XP)
